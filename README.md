 **Информационная система кинотеатра**

 **Описание проекта**
Данный проект представляет собой десктоп-приложение для комплексной автоматизации ключевых бизнес-процессов кинотеатра. Система разработана на базе фреймворка **Qt6** с использованием **C++**, что обеспечивает высокую производительность и интуитивно понятный графический интерфейс.

Целью проекта являлась разработка сбалансированного решения, сочетающего простоту использования с достаточной функциональностью для управления кинозалами, расписанием, бронированием билетов и клиентским учетом.

 **Основной функционал**
Система интегрирует следующие модули для обеспечения полного цикла управления:

**Управление фильмами:**
* добавление, редактирование и удаление фильмов (название, жанр, продолжительность, режиссер, возрастной рейтинг).

 **Управление кинозалами:**
* создание и настройка залов различной вместимости.
* визуализация схемы мест.

 **Расписание сеансов:**
* создание сеансов с привязкой к фильму, залу, дате/времени и цене.
* **интеллектуальная проверка доступности зала:** алгоритм предотвращает временные пересечения сеансов с учетом продолжительности фильма.

 **Бронирование билетов:**
* система выбора мест с **интерактивной картой зала** (цветовая индикация свободных, занятых, выбранных и VIP-мест).
* валидация бронирования (проверка доступности места, актуальности сеанса, корректности email клиента).

 **Учет клиентов:**
* регистрация клиентов и ведение истории бронирований.

**Интерфейс и удобство:**
* интуитивное управление через графический интерфейс (Qt).
* динамическая валидация вводимых данных.


 **Инструкция по запуску**
 Требования:
* **CMake:** версии 3.28 или выше.
* **Qt6:** (Core, Widgets)
* **C++ Компилятор:** с поддержкой стандарта C++17 или выше.

Сборка проекта
1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/Volkovich81/kursach.git
cd kursach

```


2. **Создайте директорию для сборки:**
```bash
mkdir build
cd build

```


3. **Настройте проект с помощью CMake:**
```bash
cmake ..

```


4. **Соберите проект:**
```bash
cmake --build .

```


5. **Запустите приложение:**
```bash
./kursach

```
